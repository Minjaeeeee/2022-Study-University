<빅데이터의 활용 7주차>
1. 신약개발 패러다임
(1) 글리벡
 → 화학물을 합성해서 약을 만들기 시작
 → Imatinib
 
(2) 신약개발 패러다임
 → 하나의 약품 하나의 타겟 하나의 질병
 → 10년에서 17년 정도 걸린다고 함
 → 그리고 10%의 성공확률을 가지고 있다

(3) 매직 불릿 
 → 1. 후보물질을 발굴하여 실험실에서 특정한 질병을 치료할 수 있는가를 판단
 → 2. 동물을 대상으로 실험을 진행한다(실험쥐)
 → 3. 임상실험을 진행한다. 무료로 약을 제공한다.(1상,2상,3상을 거침)
 → 4. 식약처에다가 판매허가 신청을 한다!(그럼 식약청에서 검증을 실시한다)
       - 미국 같은경우는 FD임(우리나라 식약청)

 (4) 결론
 → 하나의 신약 개발은 10-17년 이상 걸린다
 → 시장에 판매되는 신약 개발 성공 가능성은 10%
 → 하나의 타겟 - 하나의 약물- 하나의 질병 치유는 신약 개발의 기본 패러다임
 → 시험관 내 시험-세포시험-동물시험-임상시험-in silico


2. 신약개발의 문제점
(1) 신약개발 현장은?
 → 생물학적으로 만드는 거는 항상 증가하는데 신약은 많이 되지 않음
 → 약들이 많이 만들어내지를 못함
 → 매직불릿 아이디어는 훌륭하지만 복잡한 질병을 제대로 치유하는거에는 한계가 있다
 
(2) 생체 시스템(흐름이 이렇다는 것을 보자)
 → 실제 생체 시스템은 하위 계층의 모든 변화가 상위 계층으로 이어 지지 않는다.
 → 비만에 걸린사람은 당뇨병, 고혈압에 걸릴 가능성이 높다 (유추 가능)

(3) 결론
 → 신약개발의 성공 가능성을 높여주지는 못함(패러다임)
 → 다양한 계층과 스케일 생체 시스템의 상호작용을 통해 나타남
 → 질병 치료 예측을 높이는 것이 신약개발 성공을 거두기 위한 핵심

(4) 글로벌 제약사의 신약개발 인공지능 활용 주요 현황
 → 제약사는 여러가지 데이터들을 가지고 인공지능에서 분석을 하는 것에 특화
